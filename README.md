# autoruby

Automatically generate furigana/ruby for various document formats.

Currently planning to support:

- Markdown
- HTML
- LaTeX

Maybe:

- .DOCX
- EPUB
- MOBI
- Google Docs (WASM + extension?)

Not:

- PDF

Refer to the README file for each crate for more details.
